## Installation

To run this project, follow this step:

```
# Clone this repository
$ git clone https://github.com/dennyshuda/baliland.git

# Go intro repository
$ cd baliland

# Install dependencies
$ npm i or npm install

# Run the project
$ npm run dev
```
